## LzmaArchiveSettings.DictionarySize property

La dimensione del dizionario (buffer della cronologia) indica quanti byte dei dati non compressi elaborati di recente vengono mantenuti in memoria. Se non impostata, verrà scelta in base alla dimensione della voce.

```csharp
public int DictionarySize { get; set; }
```

### Osservazioni

Più grande è il dizionario, migliore è solitamente il rapporto di compressione, ma dizionari più grandi dei dati non compressi sono uno spreco di RAM.
