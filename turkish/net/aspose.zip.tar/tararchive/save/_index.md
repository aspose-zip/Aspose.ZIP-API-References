```csharp
public void Save(Stream output, TarFormat? format = default)
```

| output | Stream | Hedef akışı. |

### istisnalar

| istisna | şart |
| --- | --- |
| ArgumentException | *output* yazılabilir değil. - veya -*output* içinden çıkardığımız akışla aynı. - VEYA - Arşivi şuraya kaydetmek imkansız*format* biçim kısıtlamaları nedeniyle. |

### Notlar

*output* yazılabilir olmalıdır.

### Örnekler

```csharp
using (FileStream tarFile = File.Open("archive.tar", FileMode.Create))
{
    using (var archive = new TarArchive())
    {
        archive.CreateEntry("entry1", "data.bin");        
        archive.Save(tarFile);
    }
}       
```

```csharp
public void Save(string destinationFileName, TarFormat? format = default)
```

| Parametre | Tip | Tanım |
| --- | --- | --- |
| destinationFileName | String | Oluşturulacak arşivin yolu. Belirtilen dosya adı mevcut bir dosyaya işaret ediyorsa, üzerine yazılacaktır. |
| format | Nullable`1 | Tar başlık biçimini tanımlar. Boş değer, mümkün olduğunda USTar olarak değerlendirilecektir. |

### istisnalar

| istisna | şart |
| --- | --- |
| ArgumentException | *destinationFileName* sıfır uzunluklu bir dizedir, yalnızca boşluk içerir veya System.IO.Path.InvalidPathChars tarafından tanımlandığı gibi bir veya daha fazla geçersiz karakter içerir. |
| ArgumentNullException | *destinationFileName* boş. |
| PathTooLongException | Belirtilmiş*destinationFileName*, dosya adı veya her ikisi de sistem tarafından tanımlanan maksimum uzunluğu aşıyor. Örneğin, Windows tabanlı platformlarda yollar 248 karakterden ve dosya adları 260 karakterden az olmalıdır. |
| DirectoryNotFoundException | Belirtilmiş*destinationFileName*geçersiz (örneğin, eşlenmemiş bir sürücüde). |
| IOException | Dosya açılırken bir G/Ç hatası oluştu. |
| UnauthorizedAccessException | *destinationFileName* salt okunur bir dosya belirledi ve erişim Okuma değil.-veya- yol bir dizin belirledi.-veya- Arayan, gerekli izne sahip değil. |
| NotSupportedException | *destinationFileName* geçersiz bir biçimde. |

### Notlar

Bir arşivi. 'den yüklendiği yola kaydetmek mümkündür. Ancak, bu yaklaşım geçici dosyaya kopyalamayı kullandığından bu önerilmez.

### Örnekler

```csharp
using (var archive = new TarArchive())
{
    archive.CreateEntry("entry1", "data.bin");        
    archive.Save("myarchive.tar");
}       
```
