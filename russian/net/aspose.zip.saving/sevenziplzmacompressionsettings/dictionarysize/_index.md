## SevenZipLZMACompressionSettings.DictionarySize property

Размер словаря (буфера истории) показывает, сколько байт недавно обработанных несжатых данных хранится в памяти. Если не установлено, будет выбрано в соответствии с размером записи.

```csharp
public int DictionarySize { get; set; }
```

### Примечания

Чем больше словарь, тем лучше обычно степень сжатия, но словари больше, чем несжатые данные, являются пустой тратой оперативной памяти.
