## ParallelOptions.AvailableMemorySize property

Получает или задает оценку памяти в мегабайтах, доступную для размещения сжатых записей без свопинга на диск. Это значение имеет смысл, только если[`ParallelCompressInMemory`](../parallelcompressinmemory)параметр находится вAuto. Автоматический режим.

```csharp
public int AvailableMemorySize { get; set; }
```

### Примечания

Это значение используется для расчета максимального размера записи, которая может быть сжата параллельно с другими. Все записи выше рассчитанного порога будут сжаты последовательно. Безопасно иметь свойство`AvailableMemorySize`размером с свободную оперативную память и даже больше. По умолчанию предполагается, что у вас есть не менее 200 МБ на ядро ЦП.
