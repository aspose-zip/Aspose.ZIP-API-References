## SevenZipLZMACompressionSettings.DictionarySize property

字典（历史缓冲区）大小表示最近处理的未压缩数据的多少字节保留在内存中。 如果未设置，将根据条目大小进行选择。

```csharp
public int DictionarySize { get; set; }
```

### 评论

字典越大，压缩率通常就越好，但字典比未压缩数据大是浪费内存。
