## SevenZipLZMACompressionSettings.DictionarySize property

Dictionary (history buffer) size 表示最近处理的未压缩数据在内存中保留了多少字节。 如果未设置，将相应地选择条目大小。

```csharp
public int DictionarySize { get; set; }
```

### 评论

字典越大，压缩率通常越好，但比未压缩数据大的字典是对 RAM 的浪费。
